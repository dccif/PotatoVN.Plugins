# dccif's PotatoVN Plugins

欢迎来到我开发的 **[PotatoVN](https://github.com/GoldenPotato137/PotatoVN)** 的插件仓库

## 📦 包含的插件

目前本仓库包含以下插件：

### 1. 🎮 GameSaveDetector (存档自动检测器)
自动监控和检测游戏的存档目录位置。
- 两种方式的探测，文件监听器 或者 利用 ETW 实时监控文件写入
- 帮助用户快速配置游戏的存档路径

### 2. 🔗 CreateShort (快捷方式创建)
为游戏库中的游戏快速创建桌面快捷方式，或者快速导出到Sunshine进行串流。
- 一键生成游戏的快捷方式。
- 无需打开主程序即可直接从桌面启动游戏。

### 3. 📺 BigScreen (大屏幕模式)
为 PotatoVN 提供沉浸式的大屏幕/手柄操作界面。
- 适配手柄导航的 UI 设计，优化的封面展示。
- 适合在客厅电视或掌机（如 ROG Ally, Steam Deck）上使用 PotatoVN。

### 4. 📁 LanSync (局域网同步)
为 PotatoVN 提供局域网同步功能。
- 自动同步局域网中的存档
